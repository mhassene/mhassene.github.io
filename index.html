⚡ Pourquoi choisir Advanced File Search ?

Découvrez toutes les raisons qui font de notre logiciel le meilleur choix pour vos recherches de fichiers.

Ultra-simple à utiliser

Interface épurée, barre de recherche principale, résultats en temps réel. Double-clic pour ouvrir, export Excel en 1 clic. Pas de courbe d'apprentissage.

Aide intégrée (ⓘ)

Boutons d'aide contextuels partout : jokers, modes de recherche, recherche Excel par valeur, types de fichiers, pages PDF, modes Économique/Performance, et journal de débogage.

100% Portable

Un seul .exe. Lancez-le depuis une clé USB. Aucune installation, aucune clé de registre, aucun droit admin requis. Données stockées dans %USERPROFILE%\.mh_afs\.

Licence à vie

Achat unique. Pas d'abonnement, pas de renouvellement. Mises à jour mineures incluses. Utilisable sur plusieurs machines (usage personnel ou commercial).

Recherche multi-critères

Par nom (jokers), contenu (Word, Excel, PDF, Power BI), dates (création/modification), types de fichiers et extensions personnalisées.

OCR intégré Exclusif

Recherchez un mot dans un PDF scanné ou une image grâce à Tesseract. Paramétrage des pages (premières, dernières, toutes).

Recherche Excel par valeur Exclusif

Trouvez les fichiers Excel contenant des valeurs numériques dans une plage donnée. Accepte les décimaux (. ou ,) et les nombres négatifs.

Paramètres avancés

Mode Économique (faible ressource) ou Performance (vitesse). Timeout personnalisable par fichier.

Interface moderne

Palette élégante, tableau triable, barre de progression avec estimation, statistiques en direct (fichiers analysés, temps écoulé).

Gestion des dossiers

Recherche récursive, ajout/suppression de chemins, exclusion automatique des dossiers interdits (accès refusé).

Sécurité

Instance unique (pas de doublons), activation par code licence, 100% sans virus/malware.

Exportation

Exportez les résultats en Excel (.xlsx) en un clic. Journal de débogage (debug.log) pour le support technique.

Support de nombreux formats

Word (.docx, .doc) · Excel (.xlsx, .xls) · Power BI (.pbix) · PDF (texte + OCR) · Texte (.txt, .csv, .xml, .log) · Extensions personnalisées (ajoutez vos propres types).
